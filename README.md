Template within template: why "`>>' should be `> >' within a nested template argument list"  
  
位元運算子(Bitwise operators )  
因為 "<<" 是c++ 的向左移位  
">>" 是 c++ 的向右移位。  
例:  
    13>>1;  
    13<<1;  

```diff  
  vector< vector<int> > vec2d;  
# vector<(space)vector<int>(space)> vec2d;  
  
- text in red  
+ text in green  
! text in orange  
# text in gray  
@@ text in purple (and bold)@@  
```
